# Compile the shell
```bash
 make
./shell
```

# Try using the shell
Some commands to try:
```bash
    /bin/ls
    /bin/ls ..
    cd /
    /bin/pwd
    /bin/bash
    exit
    ./shell     (Note: You need to be in the project directory.)
    ./shell&    (Note: You need to be in the project directory.)
    ./b.sh      (Note: You need to be in the project directory.)
    /bin/kill -s KILL nnnn      (Where nnnn is a process ID.)
```
